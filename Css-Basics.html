CSS Basics

1. What is CSS?

CSS stands for Cascading Style Sheets. It is used to style and layout web pages written in HTML.

2. How to Connect CSS to HTML

<!-- In the <head> section -->
<link rel="stylesheet" href="style.css">

3. Basic CSS Syntax

/* Selector */
h1 {
  color: #6366f1; /* Property: Value */
  font-size: 3rem;
  text-align: center;
}

4. Important CSS Concepts

Colors & Backgrounds
color, background, background-image
Box Model
margin, padding, border
Flexbox
display: flex;
CSS Grid
display: grid;
Responsive Design
@media queries
Animations
transition, @keyframes

5. Live CSS Example

This box is styled with CSS!
.demo-box {
  background: linear-gradient(135deg, #ec4899, #8b5cf6);
  color: white;
  padding: 2rem;
  border-radius: 12px;
  text-align: center;
}
